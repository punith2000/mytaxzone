# MyTaxZone

MyTaxZone is a secure Laravel-based tax management web application with user accounts, blogs, FAQs, and case studies. It includes an admin panel with role-based access, content moderation, analytics, email notifications, subscription control, and a third-party OpenAI-powered chatbot.

## Live Demo

Frontend:
https://mytaxzone.justifytechnologies.com/

## Features

- User authentication and authorization
- Admin dashboard management
- Subscription based services
- GST and tax service modules
- Contact and enquiry management
- Responsive UI design
- Secure form handling

- PHP
- Laravel
